C언어란

c언어는 고급 언어이다.
기존에는 어셈블리 언어, 즉 저급 언어를 이용했었다.
어셈블리 언어는 하드웨어에 따라 구성이 달라져서 CPU별로 따로 개발을 했어야한다.
하지만, c언어의 등장으로 그럴 필요가 없어졌고, 심지어는 메모리에 직접 접근이 가능해졌다.

C언어의 장점

1. 절차 지향적

2. 좋은 성능: 메모리 사용, 속도 최적화하기에 유리.

C언어 특징

기본단위는 함수이다.즉, 함수를 만들고 실행 순서를 결정한다.

문장 끝에 세미콜론 ';' 을 붙인다. ex) printf("~~");

C언어 코드 예시

#include

int main() {

printf("Hello World!");

}
